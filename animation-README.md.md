# Universal Builder Animation Generator

Mission: Transform animation ideas into reusable organizational knowledge.

## Core Rules
- Never skip review gates
- Every stage must produce artifacts  
- Every decision must be traceable
- Every assumption must be explicit
- Every project contributes reusable knowledge
- Improve sources, not only outputs

## Project Structure
- universal-builder/ - Universal Builder framework integration
- src/ - Animation generator source code  
- artifacts/ - Stage output artifacts
- knowledge/ - Reusable organizational knowledge
- documentation/ - Assumptions, decisions, traceability records
